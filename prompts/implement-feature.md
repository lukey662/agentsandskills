# Implement Feature Prompt

Implement the requested feature using this workflow:

1. Confirm preserved behavior and affected layers.
2. Update Supabase schema/RLS first if data boundaries change.
3. Implement Next.js behavior with explicit server/client boundaries.
4. Apply frontend design and accessibility review.
5. Run OWASP security review.
6. Add tests proportional to risk.
7. Update living docs.

Return key changes, preserved capabilities, tests run, docs updated, and follow-up tasks.
