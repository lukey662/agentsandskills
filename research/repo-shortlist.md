# Seed Repo Shortlist

Use this as a human-reviewed starting point before automated discovery.

## Categories

- Official Next.js and Vercel examples
- Supabase official examples and reference projects
- Production SaaS apps with mature Next.js structure
- Design-system repositories with strong primitives and accessibility
- Security-heavy TypeScript repos
- Testing-heavy Next.js repos
- Repositories with strong docs, contribution guides, and agent instructions

The final 100-repo list should be produced by `agent-kit research discover` and reviewed before scanning.
