# Example Installed Output

This directory shows the expected shape after running:

```bash
npx @agent-skills/next-supabase-kit init --stack next-supabase
npx @agent-skills/next-supabase-kit audit --json
```

The example is intentionally compact. It documents the generated structure, representative manifest metadata, and expected audit shape without duplicating every template file.

The installed `.agent-kit/agent-roster.json` is the machine-readable default council contract used by agents and audited by `agent-kit audit`.
